# gflowui roadmap

## MVP

- Replace graph adapter stub with `gflow::build.iknn.graphs.and.selectk()`.
- Add full outcome + feature conditional expectation workflow.
- Integrate 3D HTML rendering from `gflow::plot3D.cont.html()` and friends.
- Add endpoint overlay flow using `gflow::geodesic.core.endpoints()`.

## V1

- Add static HTML export bundles for consortium sharing.
- Add project/session save and restore.
- Add lightweight reproducibility report (inputs, k choice, runtime parameters).

## V2

- Move long-running computation to async background jobs.
- Add hosted deployment profile and multi-user auth assumptions.
